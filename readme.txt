Dieses Plugin bindet ein Sudoku Spiel in Wordpress ein.

== Description ==

Das WordPress Plugin wp-embed-sudoku bindet auf einfachste Art und Weise ein Sudoku Spiel in einen WordPress Blog ein.

Hierzu legt man nach der Installation und Aktivierung des Plugins lediglich eine leere Seite in seinem WordPress Blog an und schreibt dort [sudoku] hinein. Fertig, schon ist Sudoku im Blog integriert.

== Installation ==

1. Unzip the ZIP plugin file
2. Copy the `WP-Findibus-Sudoku` folder into your `wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
